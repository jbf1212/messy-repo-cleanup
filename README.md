# Messy Repo Cleanup for EDS Course

## Description

This repository is for demonstrating organization of a data science project for the Environmental Data Science course at the Yale School of the Environment.

## Requirements

* Python
* Jupyter Notebooks
* Python packages:

  * pandas
  * numpy
  * matplotlib
  * seaborn

## Folder Structure

```
messy-repo-cleanup/
├─ archive/
│  ├─ analysis/
│  ├─ data/
├─ metatdata/
├─ data/
│  ├─ raw/
│  ├─ processed/
│  ├─ scripts/
├─ analysis/
│  ├─ notebooks/
│  ├─ scripts/
│  ├─ output/
│     ├─ figures
│     ├─ results
├─ notes/
```
